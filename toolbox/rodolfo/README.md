# Rodolfo Abreu's Microstate Analysis Toolbox

## Attribution

This toolbox contains MATLAB functions developed by **Dr. Rodolfo Abreu**, co-author of the research published in this repository.

**Reference:**
Abreu, R., Sim√µes, M., & Castelo-Branco, M. (2020). Pushing the Limits of EEG: Estimation of Large-Scale Functional Brain Networks and Their Dynamics Validated by Simultaneous fMRI. *Frontiers in Neuroscience*, 14, 323. https://doi.org/10.3389/fnins.2020.00323

## Contents

This toolbox provides functions for:
- Microstate analysis
- Relative amplitude computation
- Hanning window operations
- Statistical testing and outlier detection

## Acknowledgments

We gratefully acknowledge Dr. Rodolfo Abreu for his contributions to the methodology and for providing these essential analysis tools.

## Files

- `App_rs.m` - Main application for resting-state analysis
- `ComputeRelAmplitudes.m` - Relative amplitude computation
- `MicrostateAnalysis.m` - Microstate analysis functions
- `hann_window.m` - Hanning window implementation
- `hann_intersection.m` - Hanning window intersection utilities
- `myBiweight.m` - Biweight estimation
- `myFindOutliers.m` - Outlier detection
- `myMedcouple.m` - Medcouple computation
- `relAmplitude.m` - Relative amplitude utilities
- `swtest.m` - Shapiro-Wilk normality test
- `64-4_Biosemi.xyz` - Biosemi electrode coordinates
- `1-s2.0-S1388245718309015-main.pdf` - Reference paper
- `Short Tutorial - APP resting-state.docx` - Tutorial documentation
